# Data Sources

## Overview

DefenderBot content is produced using multiple source types. This page explains what those sources are and how they are used.

***

## Primary Sources

These are the most authoritative sources and are used as the foundation for all factual claims:

| Source Type | Examples |
|---|---|
| Official product websites | Bot platform homepage, feature pages |
| Official documentation | API docs, setup guides, user manuals |
| Pricing pages | Plans, tiers, limits, billing terms |
| Exchange documentation | Supported exchanges, API compatibility |
| Official announcements | Product updates, new features, changes |

***

## Secondary Sources

These sources provide industry context, competitive landscape, and broader understanding:

| Source Type | Purpose |
|---|---|
| Established review sites | Understand competitive coverage and common claims |
| Senior crypto publishers | Understand how the market leader level covers topics |
| Educational crypto resources | Provide context for beginner and intermediate readers |
| Market data sources | Bitcoin and crypto price, volatility, sentiment |
| News publications | Regulatory, exchange, and industry developments |

***

## How Secondary Sources Are Used

Secondary sources are used strictly for:
- Research awareness
- Understanding competitive angles
- Identifying missing information
- Cross-referencing claims

They are **not** copied into published pages. Published content must add original value through structure, comparison, explanation, or editorial synthesis.

***

## Verification Standard

Before publishing factual claims, priority checks include:
- Pricing verified against official pricing pages
- Exchange compatibility verified against official integration documentation
- Feature claims verified against official feature pages or documentation
- Any significant change verified against the most recent official announcement
